刚刚在Ubuntu上使用apt-cache search poppler时，发现了一个很有意思很强大很方便的PDF查看软件——apvlv，即 Alf PDF Viewer like Vim， Alf所作。

apvlv可以像操作Vim那样操作。如h,j,k,l, CTRL+F, CTRL+V, sp gg, Shift+G ,搜索/, 等等。在此一记，分享之。
可在Ubuntu上安装：sudo apt-get install apvlv
